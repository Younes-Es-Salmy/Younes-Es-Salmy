-Cette application web est développée par Younes Essalmy de TDI 202 GA (2020/2021) OFPPT (Institut Spécialisé de Technologie Appliquée Hay Al Adarissa - Fes).

(*) English :
-This web application is developed by Younes Essalmy from TDI 202 GA (2020/2021) OFPPT (Specialized Institute of Applied Technology Hay Al Adarissa - Fes).
-This web application is a chess game that can be played between tow players after setting the time for each player and if checkmate doesn't happen before ones player's time his opponent wins.
